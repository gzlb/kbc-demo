matrices module¶

Matrices Module¶

This module provides functions for matrix operations.

matrices.matrix_addition(A, B)¶

Adds two matrices A and B.

Parameters:

  • A (np.ndarray) – First matrix.

  • B (np.ndarray) – Second matrix.

Returns:

Matrix sum of A and B.

Return type:

np.ndarray

Example

>>> import numpy as np
>>> from kbc_demo.matrices import matrix_addition
>>> A = np.array([[1, 2], [3, 4]])
>>> B = np.array([[5, 6], [7, 8]])
>>> matrix_addition(A, B)
array([[ 6,  8],
       [10, 12]])
