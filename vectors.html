vectors module¶

Vectors Module¶

This module provides functions for vector operations.

vectors.vector_magnitude(v)¶

Calculates the magnitude of vector v.

Parameters:

v (np.ndarray) – Input vector.

Returns:

Magnitude of v.

Return type:

float

Example

>>> import numpy as np
>>> from kbc_demo.vectors import vector_magnitude
>>> v = np.array([3, 4])
>>> vector_magnitude(v)
5.0
